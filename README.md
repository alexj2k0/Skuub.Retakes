# Skuub Retakes

CS:GO Legacy community retakes server — configs, plugins, and deployment pipeline.

Uses SourceMod + MetaMod with custom plugins for retakes, weapons/gloves, sprays, map voting, and more.

## Contributing

1. Branch off `master`, make your changes
2. Open a PR — CI compiles all plugins to verify
3. Get approval and merge — CD deploys live to the server
